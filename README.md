# MMD Agents — Free Tier

Claude agents for product design and development, by [Multi-Modal Design](https://multimodaldesign.com).

## What's included (free)

- `/brief` — scope any feature or screen with Claude through a short structured conversation
- `decision-log` — Claude automatically captures design decisions as `DEC-NNN` records

## Install

Run this from any new project folder:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/jason-davey/mmd-agents/main/install.sh)
```

Installs into `.claude/` in your project. Works with Claude Code and Cowork.
